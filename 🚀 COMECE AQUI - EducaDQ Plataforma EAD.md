## 🆘 PRECISA DE AJUDA?

**Problemas comuns:**

1. **"Não consigo acessar a plataforma"**
   - Verifique se está usando a URL correta
   - Limpe o cache do navegador (Ctrl+Shift+Delete)
   - Tente em outro navegador

2. **"Banco de dados não conecta"**
   - Verifique se a CONNECTION STRING está correta
   - Verifique se o Supabase está online
   - Teste a conexão localmente

3. **"Email não está sendo enviado"**
   - Verifique se as credenciais do Gmail estão corretas
   - Verifique se a senha de app foi gerada corretamente
   - Verifique a pasta de spam

4. **"Deploy no Vercel falhou"**
   - Verifique os logs em https://vercel.com/dashboard
   - Verifique se todas as variáveis de ambiente estão configuradas
   - Tente fazer deploy novamente
