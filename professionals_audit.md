# Auditoria de Profissionais - Homepage

## Achados Principais

### ❌ Problemas Identificados

1. **Sem Link para Profissionais na Homepage**
   - Não há nenhum link ou menção a "Profissionais" ou "Banco de Profissionais"
   - Usuário leigo não consegue descobrir que existe uma seção de profissionais
   - Footer mostra apenas: Navegação, Redes Sociais, Contato

2. **Falta de CTA (Call-to-Action) para Profissionais**
   - Nenhum botão ou link discreto convidando profissionais a se cadastrarem
   - Oportunidade perdida de crescimento da comunidade

3. **Navegação Principal Incompleta**
   - Menu superior: Cursos, Artigos, Sobre, Contato
   - Faltam: Profissionais, Banco de Profissionais, Comunidade

### ✅ Recomendações

1. **Adicionar Link Discreto no Footer**
   - Adicionar link "Profissionais EducaDQ" no footer, seção "Navegação"
   - Texto discreto: "Conheça nossos profissionais certificados"

2. **Adicionar Seção na Homepage**
   - Seção antes do footer: "Conheça Profissionais EducaDQ"
   - Mostrar 3-4 profissionais em destaque
   - Botão "Ver Todos os Profissionais"

3. **Implementar Página de Profissionais**
   - URL: `/profissionais`
   - Busca por nome, cidade, especialidade
   - Filtros por cursos realizados
   - Cards com foto, nome, especialidade, cidade

4. **Implementar Admin Panel para Profissionais**
   - Aba "Profissionais" no AdminDashboard
   - CRUD completo: criar, editar, deletar, ativar/desativar
   - Aprovar/rejeitar cadastros
   - Ver pagamentos e status
